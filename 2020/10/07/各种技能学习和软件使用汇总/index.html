各种技能学习和软件使用汇总

2020/10/7

呦，现在开始喽!

第一类————Office系列

*Word

学习链接：Word learning

*Excel

学习链接：Excel learning

这个我墙裂推荐，别问为什么，问就是好~

*PPT和OneNote（这两个对于我们来说比较少用到，所以就你们自己去找资源啦）

第二类————计算机语言学习

*C语言

C语言作为计算机语言中的“中级语言”的存在，其重要性不言而喻。学习C语言对于计算机其他语言的学习有莫大的帮助。
不多说，附上学习链接：

C语言_鱼C小甲鱼_bilibili

这个是我当初学习C语言用的视频，我推荐这个视频主要是因为作者有趣。
大多数人是选C语言作为入门计算机科学的一门学科，所以培养兴趣是很重要的。

北京理工大学_C程序设计_上
北京理工大学_C程序设计_下

北京理工大学的计算机学科水平是位于全国前列的，学习的话也可以参考上面的两个教程。
关于C语言和C++语言进阶的学习，我会在数据结构那展开来说,当下之急，是把基础打牢。

*Python

Python是一门功能及其强大的语言，仅仅十几行的代码量，就能完成其他语言几百上千行代码量所能实现的功能。
Python语言也广泛用于大数据科学，比如数据分析及可视化（不过可视化的话，有更好的工具，比如tableau和power bi）

学习链接：
Python learning

这个视频也是小甲鱼的，因为Python相比于C语言来说，上手很容易，所以为什么不挑选更有趣的呢？

*Java

*JavaScript(HTML/CSS)

###其他语言
对于其他语言，我这里就没有给出具体地学习链接，因为世界上语言有千万种，每个语言都有自己的特点，
所以还是自己去找资源了。
p.s.可以去bilibili或者mooc，当然也可以“科学上网”，找其他地方的，我比较推荐是去他们官网学习，
因为语言更新快，官网的话，一般都会比较早更新的。

第三类————Adobe系列

*Premiere Pro

这个也简称PR，说到这个名字，你们就应该有所了解了。
PR是一个主要用于剪辑视频的软件，他功能强大，但是对电脑配置要求较高。
要想稳定流畅地运行PR，电脑的内存一般在16G以上，对显卡也要求高一些，一般是GTX1650及以上。

学习链接：
PR教程视频1
PR教程视频2

为什么我会列出两个？因为就我目前找到的PR教程，他们各有各的侧重点（因为前几天我刚去自学了PR，所以深有体会）
还有就是，学习PR是一个长期的过程，属于典型的上手容易，精通难，他需要长时间的练习，并且还要和AE、PS等软件
配合使用。但是学成之后，你就能做出一般人做不出来的转场特效，画面渲染等等。
所以，各位，加油吧！

*PhotoShop

简称PS，主要是用于修图和制作海报等（还有抠图，抠出来的图用到视频剪辑里，抠图老费头发了）。

学习链接：
PS learning速成版
PS learning详细版

这上面的两个版本，你们按自己需求学习。

*Dreamweaver

简称DW，这是一个与网页制作有关的软件，他的语言基础是HTML、CSS和JavaScript。

学习链接：
DW learning

没啥学习链接，上手极其简单，几分钟就可以

第四类————计算机专业必学课程

*数据结构

数据结构在计算机科学中，我个人认为是最重要的，原因有以下几点：

1.他能使我们写的代码分块，并且可以把自己需要的函数放到自定义的头文件中，以便其他用户使用；
2.数据结构和算法是密不可分的，算法大可理解为解决问题的办法，其中就包括最优解的问题；
3.在数据结构时，我们还会接触到C++的类或者模板的概念，这几个概念对于C++的深入学习尤为重要。

学习链接：
普林斯顿大学 算法_第一部分
斯坦福大学 算法分析与设计
（小甲鱼）数据结构和算法

数据结构的学习是需要付出大把时间的，不是仅仅就看几个视频就能掌握的，
所到底，计算机专业是一个工科性质与理科性质并重的，所以花在实践上的时
间不可能比看视频的时间少，甚至还要多许多。

*数据库

现在是大数据时代，数据库也随之大放光彩，其实不仅仅在计算机学科上，
在其他领域比如金融等需要处理非常多数据的行业，他们业内的专业人士
也或多或少的会去学习数据库的相关知识，本质上讲，数据库也算一门计算
机语言，即SQL。

学习链接：
数据库系统：模型与语言
数据库系统：建模与设计
数据库系统：管理与技术

*计算机网络

计网是比较难学的一门，目前应用的我个人觉得不多，但是不代表他不重要
我们上网是有一个TCP/IP协议的，它就是计网的一个部分，还有很多比如电脑
防火墙、电脑硬件配置等，都需要相应地掌握计算机网络的知识并加以应用。

学习链接：
计算机网络
斯坦福大学 CS144

*计算机组成原理

这门课程是偏向于硬件方面了，学习计算机组成原理，可以很好地帮助我们
理解计算机是怎么由几个电路和几个芯片等部件组成，最终能够完成复杂命令的

学习链接：
计算机组成原理
